# Contributing to Awesome Cybersecurity for Middle School

First off, thank you for taking the time to contribute! 🎉

This project aims to be the go-to resource hub for educators teaching cybersecurity — especially AP Cybersecurity — at the middle school and early high school level.

## How to Contribute

### Adding a New Resource

1. **Fork** this repository
2. **Create a branch** for your addition: `git checkout -b add-new-resource`
3. **Add your resource** to the appropriate section in `README.md` (or a sub-document)
4. **Follow the table format** already used in the document
5. **Commit** with a clear message: `git commit -m "Add [resource name] to [section]"`
6. **Open a Pull Request** with a description of what you added and why

### What We're Looking For

- ✅ **Interactive tools** — hands-on, student-friendly platforms and demos
- ✅ **CTF platforms** — especially beginner-friendly ones suitable for younger students
- ✅ **Curriculum resources** — textbooks, course platforms, lesson plans
- ✅ **Unit-specific resources** — materials aligned to specific AP Cybersecurity topics
- ✅ **Teaching strategies** — pedagogical approaches, pacing tips, classroom activities
- ✅ **Free or low-cost resources** — priority given to resources accessible to all schools

### What We're Not Looking For

- ❌ Paid tools without a free tier suitable for classroom use
- ❌ Resources that require advanced prerequisites beyond middle school level
- ❌ Duplicates of existing entries
- ❌ Resources behind paywalls without educational access

### Formatting Guidelines

- Use the **table format** consistent with the existing README
- Include a **brief description** of what the resource is and how it's useful
- Add the **direct link** to the resource
- If applicable, note the **difficulty level** (Beginner / Intermediate / Advanced)
- Keep descriptions **concise** — one to two sentences

### Suggesting Improvements

Found a broken link? Notice outdated information? Have an idea for a new section?

1. Open an **Issue** describing the problem or suggestion
2. Or submit a **Pull Request** with the fix

### Review Process

All pull requests will be reviewed for:
- Relevance to middle school / early high school cybersecurity education
- Accuracy of descriptions and links
- Formatting consistency
- Appropriate licensing (we prefer freely accessible resources)

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.

## Questions?

Feel free to open an issue with any questions about contributing. We're happy to help!
