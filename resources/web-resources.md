# Web Resources

A curated list of web resources collected during APSI training, organized by category.

---

## Interactive Learning Platforms

| Resource | URL | Description |
|---|---|---|
| Hacksplaining | [hacksplaining.com/lessons](https://www.hacksplaining.com/lessons) | Real-time, interactive demos of common cyberattacks (XSS, SQL injection, CSRF). Very intuitive and student-friendly. |
| Google Gruyere | [google-gruyere.appspot.com](https://google-gruyere.appspot.com/) | Codelab for web security vulnerabilities — XSS, XSRF. Students exploit a vulnerable app. |
| CyberChef | [cyberchef.org](https://cyberchef.org/) | The "Cyber Swiss Army Knife" — encode, decode, encrypt, hash, analyze data. |
| Blockchain Hash Demo | [andersbrownworth.com/blockchain/hash](https://andersbrownworth.com/blockchain/hash) | Real-time visual demo of cryptographic hashing. |
| Cryptosystem Demo | [kerryveenstra.com/cryptosystem.html](https://www.kerryveenstra.com/cryptosystem.html) | Simplistic symmetric & asymmetric key encryption. Short keys, weak but straightforward. |
| KEJSON Symmetric Encryption | [kejson.com/en/encrypt/symmetric](https://www.kejson.com/en/encrypt/symmetric/) | Interactive symmetric encryption demo. |
| KEJSON (main) | [kejson.com/en](https://www.kejson.com/en/) | KEJSON cybersecurity tools hub. |
| Have I Been Pwned | [haveibeenpwned.com](https://haveibeenpwned.com/) | Check if email/accounts appear in known data breaches. |
| Padlet Arcade | [arcade.padlet.com](https://arcade.padlet.com) | Create simple educational games and interactive boards. |
| Padlet | [padlet.com](https://padlet.com) | Collaborative boards for classroom activities. |
| Cyber Case Files | [nascar-paul.github.io/cyber-case-files](https://nascar-paul.github.io/cyber-case-files/) | Interactive cyber case files (GitHub Pages). Also on [Netlify](https://super-sunshine-f3134d.netlify.app/). |
